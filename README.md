### Future Projects
- `central-bank-sentiment-tracker`: API script that parses through Fed policy statements and delivers percentage estimate of policy change. Backtested against interest rate cycles.
- `global-market-brief`: An automated script retrieving data across international equity indices, calculating percentage changes, and utilizing webhook notifications to deliver market summaries.
- `msm-portfolio-allocation`: A Markov-Switching model that identifies macro regimes and optimizes a cross asset portfolio weight. Backtested against a 60/40 portfolio.
